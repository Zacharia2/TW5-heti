# TW5-heti 赫蹏（hètí）

专为中文内容展示设计的排版样式增强。它基于通行的中文排版规范而来，可以为网站的读者带来更好的文章阅读体验。


简单的说，可以在单文件版tw使用不出问题，但是在nodejs版tw中使用就会出现这种情况。
